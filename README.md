# To do list tracker
A to do list tracker built with React, Vite and CSS.

## Features
- Add To do tasks to track
- Mark completed tasks using a checkbox input
- Delete old tasks
- Data persists with localStorage

- usings hooks like useState and useEffect
- How data is stored and how to handle data in react
